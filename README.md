# Formatação de Texto

```css
    color: ;
```

* ***color***

> Muda a cor do texto.

```css
    font-size: ;
```

* ***font-size***

> Muda o tamanho da fonte.

```css
    font-family: ;
```

* ***font-family***

> Muda o tipo da fonte.

```css
    font-weight: ;
```

> Muda a espessura da fonte.

```css
    text-align: ;
```

> Alinha o texto na página, *div* e etc;.

```css
    text-transform: ;
```
